# Статус готовности историй

Статус: завершено

- US-01: Высокий
  - История имеет явный источник, самостоятельную ценность и проверяемые критерии для отображения статуса и прогресса активной цели.
- US-02: Высокий
  - История изолирует создание цели со шагами и фиксирует правило расчета прогресса без ручного ввода процентов.
- US-03: Высокий
  - История покрывает отдельный пользовательский результат: выполнение шага, автоматический пересчет прогресса и переход в `в процессе`.
- US-04: Высокий
  - История четко ограничена ручным управлением рабочими статусами и не смешивает его с завершением цели.
- US-05: Высокий
  - История фиксирует условия завершения цели со шагами, ее выход из активного списка и перенос в достижения без потери трассируемости.
- US-06: Высокий
  - История самостоятельна, отделяет завершение без шагов от удаления и содержит проверяемые критерии по достижениям.
