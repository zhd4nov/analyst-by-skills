# Уточненные требования

Статус: завершено

Акторы
- Пользователь приложения по постановке личных целей

Цели
- Понимать, на каком этапе находится каждая цель
- Видеть, есть ли продвижение по цели
- Завершать цель без ее удаления

Текущий функциональный объем
- Добавить учет прогресса по цели
- Показать текущее состояние цели пользователю
- Добавить явное завершение цели

Ключевые сценарии
- пользователь открывает список или карточку целей и видит текущее состояние цели
- пользователь фиксирует, что по цели есть продвижение
- пользователь помечает цель как завершенную

Известные ограничения
- В текущем состоянии приложения цель можно только создать или удалить
- Процент прогресса нельзя задавать вручную
- Если шаги не добавлены при создании, прогресс цели дискретный: `0%` или `100%`
- Цель в статусе `завершена` видна в разделе достижений
- Цель в статусе `завершена` нельзя открыть снова
- Удаление цели означает отказ от цели, а не ее достижение
- При наличии шагов цель завершается автоматически после выполнения всех шагов и подтверждения пользователя
- Если шагов нет, цель завершается после явной отметки выполнения пользователем
- После отказа завершить цель при выполнении последнего шага пользователь должен иметь возможность добавить новые шаги, а прогресс должен быть пересчитан
- Первый выполненный шаг автоматически переводит цель из `черновик` в `работаем`
- Пользователь может вручную перевести цель из `черновик` в `работаем`
- Статус `пауза` устанавливается только вручную пользователем
- Возврат цели в статус `черновик` не допускается
