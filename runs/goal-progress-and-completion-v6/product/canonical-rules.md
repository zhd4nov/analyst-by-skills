## Правило
ID: CR-03
Тип: бизнес-правило
Формулировка: процентный прогресс применяется только к цели в статусе `в процессе`, если цель состоит из шагов.
Источник: Лог уточнений: CL-01
Использовать дальше:
- Спецификация -> Бизнес-правила
- Пользовательские истории

## Правило
ID: CR-04
Тип: валидация
Формулировка: процент прогресса цели со шагами рассчитывается на основе выполненных шагов внутри цели.
Источник: Лог уточнений: CL-01
Использовать дальше:
- Спецификация -> Валидационные правила
- Пользовательские истории

## Правило
ID: CR-05
Тип: переход состояния
Формулировка: после выполнения всех шагов система должна спросить пользователя, достигнута ли цель или нужно добавить еще шаги.
Источник: Лог уточнений: CL-02
Использовать дальше:
- Спецификация -> Модель состояний / Переходы
- Пользовательские истории

## Правило
ID: CR-06
Тип: переход состояния
Формулировка: если пользователь подтверждает достижение цели после выполнения всех шагов, цель переводится в статус `завершена`.
Источник: Лог уточнений: CL-02

- Спецификация -> Модель состояний / Переходы
