# Уточненные требования

Статус: complete

## Акторы
- Пользователь приложения по постановке личных целей. Источник: Вход.

## Цели
- Пользователь должен понимать состояние жизненного цикла цели. Источник: Вход; [Лог уточнений: CL-01].
- Пользователь должен видеть прогресс достижения цели, если цель находится в процессе и состоит из шагов. Источник: Вход; [Лог уточнений: CL-01].
- Пользователь должен завершать достигнутые цели и видеть их отдельно от активных целей. Источник: Вход; [Лог уточнений: CL-02].
- Пользователь должен работать с простыми целями, которым не нужны шаги. Источник: [Лог уточнений: CL-03].

## Текущий функциональный объем
- Учет статуса цели: черновик, в процессе, завершена. Источник: [Лог уточнений: CL-01].
- Расчет процентного прогресса для цели в процессе на основе выполненных шагов-задач. Источник: [Лог уточнений: CL-01].
- Подтверждение завершения цели со шагами после выполнения всех шагов. Источник: [Лог уточнений: CL-02].
- Добавление дополнительных шагов перед завершением цели и пересчет прогресса. Источник: [Лог уточнений: CL-02].
- Простая цель без шагов с бинарным состоянием выполнена / не выполнена и ручным завершением. Источник: [Лог уточнений: CL-03].
- Редактирование цели в процессе, добавление шагов и удаление невыполненных шагов. Источник: [Лог уточнений: CL-03]; [Лог уточнений: CL-05].
- Перемещение завершенной цели из активных целей в список достижений. Источник: [Лог уточнений: CL-02].
- Округление процента прогресса до целого числа. Источник: [Лог уточнений: CL-04].
- Запрет удаления выполненного шага. Источник: [Лог уточнений: CL-05].
- Перевод цели из `черновик` в `в процессе` после выполнения первого шага или ручного действия "начать". Источник: [Лог уточнений: CL-06].

## Ключевые сценарии
- Пользователь создает цель и выбирает, будет ли она простой или будет состоять из шагов. Источник: [Лог уточнений: CL-03].
- Пользователь видит статус цели и, для цели со шагами в процессе, процент прогресса. Источник: Вход; [Лог уточнений: CL-01].
- Пользователь отмечает шаги как выполненные, и прогресс цели пересчитывается. Источник: [Лог уточнений: CL-01].
- После выполнения первого шага цель переходит из `черновик` в `в процессе`, если она еще была черновиком. Источник: [Лог уточнений: CL-06].
- Пользователь может вручную начать цель через действие "начать". Источник: [Лог уточнений: CL-06].
- Когда все шаги выполнены, система уточняет, достигнута ли цель или пользователь хочет добавить шаги. Источник: [Лог уточнений: CL-02].
- Если пользователь подтверждает достижение, цель становится завершенной и попадает в список достижений. Источник: [Лог уточнений: CL-02].
- Если пользователь добавляет шаги вместо подтверждения, система пересчитывает процент прогресса. Источник: [Лог уточнений: CL-02].
- Пользователь вручную завершает простую цель. Источник: [Лог уточнений: CL-03].

## Известные ограничения
- Сейчас цель можно только поставить и удалить. Источник: Вход.
- Завершенная цель является терминальным статусом; возврат назад невозможен. Источник: [Лог уточнений: CL-03].
- Пока цель находится в процессе, ее можно отредактировать, добавить новые шаги или удалить невыполненные старые шаги. Источник: [Лог уточнений: CL-03]; [Лог уточнений: CL-05].
- Выполненный шаг удалить нельзя. Источник: [Лог уточнений: CL-05].
